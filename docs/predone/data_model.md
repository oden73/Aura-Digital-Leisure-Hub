# Классификация мультимедиа

## Общие критерии

| Название | Пример | Примечание |
|-----------|----------|-------------|
| Жанр | Драма, комедия, хоррор, фантастика | Это первое, на что ориентируется человек при выборе |
| Сеттинг / Мир | Исторический, футуристический, фэнтези, современный мир | Время и место действия, создающие атмосферу |
| Тематика и проблематика | Смерть, любовь, политика, взросление | Глубокий смысл и идеи автора |
| Тональность | Мрачная, легкая, ироничная, эпическая | Общее настроение |
| Целевая аудитория | 6+, 12+, 18+ | Границы допустимого контента и направленность на конкретного зрителя/игрока |
